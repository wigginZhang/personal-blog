# 博客图片上传系统设计

**日期：** 2026-04-18
**项目：** personal-blog
**状态：** 设计中

## 背景

博客使用 Astro 构建，文章以 Markdown 格式存储在 `src/content/articles/`。通过 `polish-article.ts` 脚本润色并发布博客。需要支持：

1. 润色时粘贴图片 → 自动保存并插入链接
2. 对已有文章追加图片

## 功能需求

### 场景 1：润色时插入图片

**流程：**
1. 用户输入原始文字（含或不含图片标记指令）
2. 复制图片到剪贴板
3. 运行 `npm run polish`
4. AI 润色文案
5. 脚本检测剪贴板图片 → 保存到 `public/images/`
6. 替换标记或追加到末尾
7. 复制图片链接到剪贴板
8. 发布博客

**标记指令：**
- `需要图1` → 插入第一张图片
- `需要图2` → 插入第二张图片
- 无标记 → 所有图片追加到文章末尾

**示例输入：**
```
[旺柴]懒惰是人类进步的阶梯...-需要图1-早知道直接说精通了[破涕为笑]
```

**输出：**
```markdown
[旺柴]懒惰是人类进步的阶梯...
![image](./images/2026-04-18-143052-abc123.png)
早知道直接说精通了[破涕为笑]
```

### 场景 2：保存后追加图片

**流程：**
1. 用户运行 `img add`（或 `npm run img add`）
2. 显示文章列表（命令行列表选择）
3. 用户选择目标文章
4. 粘贴图片
5. 图片链接追加到文章末尾
6. 复制链接到剪贴板

## 技术方案

### 文件结构

```
personal-blog/
├── public/
│   └── images/              # 统一图片存储
│       ├── 2026-04-18-143052-abc123.png
│       └── 2026-04-18-144530-def456.png
├── scripts/
│   ├── img.ts                # 图片工具主文件
│   │   ├── paste             # 粘贴上传（场景1内部使用）
│   │   ├── add               # 添加到已有文章
│   │   └── list              # 列出已有图片
│   └── polish-article.ts     # 集成图片功能
└── src/
    └── content/
        └── articles/         # 博客文章
```

### 图片命名

格式：`{年}-{月}-{日}-{时分秒}-{6位随机字符}.png`

示例：`2026-04-18-143052-a3f1b2.png`

### 图片链接格式

生成相对路径：`![image](./images/{文件名})`

### 场景 2：img add 命令

**插入位置逻辑：**
- md 中**没有**图片链接 → 新图片插入到**文章末尾**
- md 中**已有**图片链接 → 新图片插入到**最后一个图片链接的后面**

**交互流程：**
```
$ img add

📚 选择文章：

  1. hello-world
  2. the-lazy-persons-guide-to-excel-mastery
  3. another-article

选择文章 (输入编号): 2

📷 请粘贴图片 (Ctrl+V)...

📷 图片已保存: public/images/2026-04-18-143052-abc123.png
🔗 已插入到文章末尾
🔗 已复制图片链接到剪贴板
```

**多张图片：** 依次粘贴，连续插入到目标位置

## 命令行接口

```bash
# 场景1：润色时粘贴图片（集成到 polish）
npm run polish

# 场景2：保存后追加图片
img add              # 交互式选择文章
img add <文件名>     # 直接指定文章
img list             # 列出已有图片
img rm <文件名>       # 删除图片
```

## 错误处理

| 场景 | 处理 |
|-----|------|
| 剪贴板无图片 | 正常润色，不提示 |
| 图片保存失败 | 报错，询问是否重试 |
| 标记的文章无对应图片 | 标记保留原文 |
| 选择不存在的文章 | 报错，重新选择 |
| Git 发布失败 | 文件已保存，提示手动发布 |
